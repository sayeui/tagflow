# 文件管理器 API 文档

## 概述
本文档提供了文件管理器 API 的详细描述，包括上传文件、管理标签和导入文件夹的端点。

## 端点

### 1. 上传文件
- **端点:** `/upload`
- **方法:** `POST`
- **描述:** 将文件上传到对象存储。
- **请求:**
  - **表单数据:**
    - `file` (必需): 要上传的文件。
- **响应:**
  - **成功 (200):**

### 2. 获取文件标签
- **端点:** `/tags/<path:file_path>`
- **方法:** `GET`
- **描述:** 获取文件的标签。
- **响应:**
  - **成功 (200):**
    ```json
    {
      "tags": ["标签1", "标签2"]
    }
    ```

### 3. 获取文件内容
- **端点:** `/files/<path:object_name>`
- **方法:** `GET`
- **描述:** 从对象存储中获取文件内容。
- **响应:**
  - **成功 (200):**
    ```json
    {
      "content": "文件内容"
    }
    ```
  - **错误 (500):**
    ```json
    {
      "error": "错误信息"
    }
    ```

### 4. 导入文件夹
- **端点:** `/folders/import`
- **方法:** `POST`
- **描述:** 将文件夹中的所有文件导入到对象存储。
- **请求:**
  - **表单数据:**
    - `folder_path` (必需): 要导入的文件夹路径。
- **响应:**
  - **成功 (200):**
    ```json
    {
      "message": "文件夹导入成功"
    }
    ```

### 5. 添加文件标签
- **端点:** `/tags`
- **方法:** `POST`
- **描述:** 为文件添加标签。
- **请求:**
  - **表单数据:**
    - `file_path` (必需): 文件路径。
    - `tags` (必需): 标签列表。
- **响应:**
  - **成功 (200):**
    ```json
    {
      "message": "标签添加成功"
    }
    ```

### 6. 删除文件标签
- **端点:** `/tags`
- **方法:** `DELETE`
- **描述:** 从文件中删除标签。
- **请求:**
  - **表单数据:**
    - `file_path` (必需): 文件路径。
    - `tags` (必需): 标签列表。
- **响应:**
  - **成功 (200):**
    ```json
    {
      "message": "标签删除成功"
    }
    ```
